# 自我介绍
我是西安电子科技大学软件工程专业的本科生吕富民。从大一起，我就对软件开发和开源技术产生了浓厚兴趣。曾参与过校内“星火杯”竞赛和大学生创新创业训练计划（大创），期间与学长合作完成了一个基于 Flutter 的简易界面设计开源项目，初步体验了团队协作与代码开源的过程。 

日常学习中，我习惯使用 GitHub 上的开源工具（如文档生成器、代码模板、CLI 工具等）来提升效率，并已将部分课程作业（如《程序设计基础》《面向对象程序设计》）整理后上传至个人 GitHub 仓库，作为学习记录与技术积累。此外，我还参加过书院组织的 Hugo 静态网站开发实践，对前端构建与版本管理有了初步了解。 

虽然我经常使用 GitHub，也维护过自己的小项目，但尚未真正参与过社区级的开源项目贡献。希望通过本课程，迈出“从使用者到贡献者”的关键一步。 

# 对本课程的期待

## 我希望通过这门开源实践课程： 

- 系统学习 GitHub 的协作工作流，尤其是 Pull Request、Code Review、Issue 管理等核心流程；
- 在真实或模拟的开源场景中锻炼工程能力，提升代码质量意识和文档撰写习惯；
- 与小组同学共同完成一次完整的开源项目实践，积累团队协作经验；
- 最重要的是，完成人生第一个对外部开源项目的有效贡献（哪怕是修复一个拼写错误或完善一行文档），真正融入开源社区文化。
     
我相信，开源不仅是技术共享，更是一种学习和成长的方式。这门课将是我走向更广阔开发者社区的重要起点。 

## 感兴趣的开源项目领域或方向 

目前我对以下方向的开源项目比较感兴趣： 

- 开发者工具类：如命令行工具（CLI）、代码生成器、学习辅助脚本等，这类项目贴近日常开发，容易上手；
- 教育/文档类项目：例如开源教程、技术文档翻译、学习资源整理等，既能帮助他人，也能巩固自身知识；
- 轻量级前端框架或组件库：受之前 Flutter 和 Hugo 实践的影响，我对用户界面相关的开源项目有一定兴趣；
- 高校或学生主导的开源项目：这类项目通常对新手友好，社区氛围包容，适合初次贡献者。
     
未来也希望在掌握基础后，逐步尝试参与更广泛的开源生态，在更多项目中提高自己的编程能力与对开源社区的贡献。
